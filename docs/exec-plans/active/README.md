# Active ExecPlans

Place active ExecPlans in this directory.

Each ExecPlan must follow `/hyperopen/.agents/PLANS.md` and remain a self-contained living document with updated progress, discoveries, decision log, and retrospective sections.

Active means currently executing. Every file here must reference live `bd` work and must retain at least one unchecked progress item. When the work is complete, move the plan out of this directory immediately. `npm run check` enforces these guardrails.
