# Active ExecPlans

Place active ExecPlans in this directory.

Each ExecPlan must follow `/hyperopen/.agents/PLANS.md` and remain a self-contained living document with updated progress, discoveries, decision log, and retrospective sections.

Active means currently executing. Every file here must retain at least one unchecked progress item and include durable context references when applicable. Beads / `bd` references are optional local scratch only and do not satisfy public work tracking by themselves. When the work is complete, move the plan out of this directory immediately. `npm run check` enforces these guardrails.
